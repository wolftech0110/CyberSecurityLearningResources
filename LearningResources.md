## Learn Cybersecurity by starting with these sites.
Here are some sites that I would review to get started learning the basics of Cybersecurity.
- TryHackMe - https://tryhackme.com/
- HackTheBox academy - https://academy.hackthebox.com/
- TCM Security - https://academy.tcm-sec.com/

## Blue Team (Defensive Security)
- Letsdefend.io - https://letsdefend.io/
- TheHive: Open Source Incident Response Platform - https://github.com/TheHive-Project/TheHive
- Wazuh - The Open Source Security Platform. - https://github.com/wazuh/wazuh

- Reverse Engineering - https://malwaretech.com/labs/
