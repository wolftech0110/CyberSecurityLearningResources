# Cybersecurity Learning Resources

This repository contains a curated collection of cybersecurity learning resources collected by **wolftech0110**.

The purpose of this repository is to provide a practical, structured starting point for learning cybersecurity, covering foundational knowledge, defensive security (Blue Team), offensive security (Red Team), and general security research.

All learning links and references are maintained in **LearningResources.md**.

---

## Repository Structure

- `LearningResources.md`  
  The primary resource file containing links to:
  - Beginner cybersecurity learning platforms
  - Blue Team (defensive security) resources
  - Red Team (offensive security) resources
  - General security reading and research material

---

## How to Use This Repository

1. Start with the beginner resources in `LearningResources.md` if you are new to cybersecurity.
2. Explore Blue Team or Red Team sections based on your interests or career goals.
3. Use the general security resources to deepen understanding and explore historical and advanced topics.
4. Combine reading with hands-on labs whenever possible.

This repository is intended to be a living document. Resources may be expanded, reorganized, or updated over time.

---

## Intended Audience

- Beginners learning cybersecurity fundamentals
- SOC analysts and defensive security practitioners
- Penetration testers and offensive security learners
- Anyone interested in hands-on, ethical security research
